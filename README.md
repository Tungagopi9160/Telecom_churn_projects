# Telecom Customer Churn Prediction

## Overview

This project builds a Machine Learning model to predict whether a telecom customer is likely to churn (leave the service). It uses customer demographic and service data to help businesses identify at-risk customers and improve customer retention.

## Features

* Data preprocessing and cleaning
* StandardScaler for numerical feature scaling
* Label Encoding for categorical features
* Feature extraction and train-test split
* Logistic Regression model training
* Churn prediction and accuracy evaluation
* Feature importance visualization using Matplotlib

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Workflow

1. Load the telecom customer churn dataset.
2. Preprocess numerical and categorical features.
3. Scale numerical data and encode categorical variables.
4. Split the dataset into training and testing sets.
5. Train a Logistic Regression model.
6. Predict customer churn on test data.
7. Evaluate model performance using accuracy score.
8. Visualize feature importance.

## Results

The model successfully predicts customer churn using customer-related features, providing insights that can support customer retention strategies.
